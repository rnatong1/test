# 손바닥 표정 게임 프로토타입

## 실행

카메라 권한 때문에 `file://`로 직접 열기보다 로컬 서버로 실행하는 걸 권장합니다.

예:

```bash
npx serve
```

또는 VS Code의 Live Server 같은 도구를 사용해도 됩니다.

## 동작

- 카메라 시작 버튼을 누른다
- 손바닥을 카메라에 보여준다
- 손바닥이 인식되면 캐릭터 표정이 랜덤으로 바뀌고 점수가 오른다

## 메모

- 현재 버전은 프로토타입이라 캐릭터는 SVG 기반의 간단한 얼굴입니다
- 이후 실제 캐릭터 이미지로 바꾸기 쉽게 구조를 단순하게 유지했습니다
